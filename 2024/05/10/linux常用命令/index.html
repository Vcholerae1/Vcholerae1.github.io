linux常用命令 & git常用指令

Author: V.cholerae Date: May 10, 2024  23:54:53 Category: linux

一些linux常用命令

  1. ctrl c: 取消命令，并且换行
  2. ctrl u: 清空本行命令
  3. tab键：可以补全命令和文件名，如果补全不了快速按两下tab键，可以显示备选选项
  4. ls: 列出当前目录下所有文件，蓝色的是文件夹，白色的是普通文件，绿色的是可执行文件
  5. pwd: 显示当前路径
  6. cd XXX: 进入XXX目录下,
    cd .. 返回上层目录
    cd /：切换到系统根目录
    cd ~：切换到用户主目录
    cd -： 切换到上一个操作所在目录
  7. cp XXX YYY: 将XXX文件复制成YYY，XXX和YYY可以是一个路径，比如../dir_c/a.txt，表示上层目录下的dir_c文件夹下的文件a.txt
  8. mkdir XXX: 创建目录XXX
  9. rm XXX: 删除普通文件; rm XXX -r: 删除文件夹
  10. mv XXX YYY: 将XXX文件移动到YYY，和cp命令一样，XXX和YYY可以是一个路径；重命名也是用这个命令
  11. touch XXX: 创建一个文件
  12. cat XXX: 展示文件XXX中的内容
  13. 复制文本 Ctrl + insert
  14. 粘贴文本 Shift + insert # tmux 的使用 tmux 是一个在类unix系统下经常使用的终端复用工具 # vim的使用 vim是常用的文本编辑器 常用命令
    （1） i进入编辑模式 # git 基本知识及指令 git 是一个流行的代码版本管理分布式框架
